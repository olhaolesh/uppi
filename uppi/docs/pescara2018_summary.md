# Accordo Territoriale Pescara 2018 – технічне резюме для розрахунку canone

Цей документ — технічний конспект Accordo Territoriale Comune di Pescara (2018)
для подальшої реалізації в коді (uppi.domain.pescara2018_*).

Тут немає юридичних тлумачень, лише:
- відповідність microzona/foglio → ZONA;
- типології та діапазони площ;
- базові діапазони €/mq по зонах і subfasce;
- правила застосування елементів A/B/C/D;
- відсоткові корекції (мебльований, клас енергоефективності, студенти);
- загальні ліміти варіацій, якщо передбачені.
